# Bot Trading Gemini

Bot Telegram untuk analisis trading futures otomatis menggunakan AI Gemini dan data CoinGecko.

## Cara Pakai
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Jalankan bot:
   ```
   python main.py
   ```
3. Gunakan perintah /start di Telegram.

## Struktur Folder
- main.py
- handlers/
- utils/
- config.py
- requirements.txt
- README.md
